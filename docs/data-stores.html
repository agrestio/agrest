Chapter 3. Supported Data Stores

LinkRest is written on top of Apache Cayenne ORM and supports any relational data store, such as MySQL or Oracle. Support for alternative data stores can be provided with relative ease. E.g. NoSQL databases, flat files, etc. E.g. we've written POJO and LDAP backends.
