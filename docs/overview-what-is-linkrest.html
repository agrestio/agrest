Chapter 1. What is LinkRest

LinkRest is a set of tools to build web APIs for your data. You would use it to expose a data store (such as a relational database) as a RESTful web service. To achieve that goal LinkRest defines a simple HTTP/JSON-based protocol (described in Part II, “LinkRest Protocol”) and a server-side framework (described in Part III, “LinkRest Framework”). The protocol ensures that any client, be it JavaScript, a mobile app or maybe a server-side app, can make sense of any LinkRest-enabled service. The framework is written in Java and helps you to build services adhering to the protocol.

LinkRest is open source and is distributed under the terms of BSD license
