Chapter 10. Prerequisites

  • Java 1.7 or newer

  • A Java "war" project that will serve your REST requests.

  • A JAX-RS 2.0 container, such as Jersey 2.x.

  • Cayenne 4.0.M2 or newer. Mapping your database and starting Cayenne ServerRuntime is outside the scope of this document. Please refer to the corresponding Cayenne docs.
